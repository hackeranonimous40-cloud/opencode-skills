## 🚀 Despliegue Rápido

```bash
# Clonar arsenal
git clone https://github.com/hackeranonimous40-cloud/opencode-skills.git

# Copiar a tu instancia de OpenCode
cp -r opencode-skills/* ~/.opencode/skills/

# Activar y dominar
opencode --load-skills
```
